# 💰 Money-Radar-Pro

> A high-performance financial intelligence dashboard.

Money-Radar-Pro is a full-stack MERN application (Next.js 15 + Fastify + MongoDB) designed to track, analyze, and predict your spending using AI.

## 🏗️ Architecture
- **Frontend:** Next.js (App Router), Tailwind CSS, TypeScript.
- **Backend:** Fastify (Ultra-fast Node.js framework), Zod, MongoDB.
- **AI:** Gemini Pro integration for financial insights.

## 🚀 Quick Start

### Backend Setup
```bash
cd backend
npm install
npm run start
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
